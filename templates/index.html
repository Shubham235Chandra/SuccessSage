End to End Machine Learning Project: Success Sage - A Student Exam Performance Indicator and Predictor

About This Project

This project demonstrates a complete machine learning pipeline, from data ingestion to model training and evaluation. The pipeline is implemented in Python and uses various libraries such as scikit-learn, pandas, and XGBoost.

Project Components

  • Data Ingestion: Reads raw data from CSV files, splits it into training and testing sets, and saves them for further processing.
  • Data Transformation: Applies preprocessing steps to the data, such as scaling numerical features.
  • Model Training: Trains multiple machine learning models using hyperparameter tuning and evaluates their performance.

Technologies Used

  • Python
  • pandas
  • scikit-learn
  • XGBoost
  • Pickle

How to Run the Project

  1. Ensure you have Python and the necessary libraries installed.
  2. Run the data ingestion script to load and split the data.
  3. Run the data transformation script to preprocess the data.
  4. Run the model trainer script to train and evaluate the models.

Project Files

  • data_ingestion.py: Handles the reading and splitting of data.
  • data_transformation.py: Manages data preprocessing.
  • model_trainer.py: Trains and evaluates machine learning models.
  • utils.py: Contains utility functions for saving and loading models, as well as model evaluation.
  • exception.py: Custom exception handling for better error management.
  • logger.py: Configures logging for tracking the execution of the scripts.
